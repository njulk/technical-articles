模型

```
+-------------------+      +--------------------+
| SELECT c1, c2, c3 |      |  F1: column1,      |
| FROM t1 ...       |      |  F2: column2 ...   |
+----------+--------+      +--------^-----------+
           |                        |
           |                        |
           |                Copy value for every column
           |                        |
           |                        |
+----------v------------------------+-----------------------------------+
|                              database/sql                             |
+----------+------------------------^-----------------------------------+
           +                        +
    Prepare Query              return []driver.Value
           +                        +
+----------v------------------------+-----------------------------------+
|                              MySQL Driver                             |
+----------+------------------------^-----------------------------------+
           |                        |
           |                        +
           +                    Read the input stream
    Exec Query                      +
           +                        |
           |                        |
           |                  +-----+------+
           |                  |            |
           +------------------>   MySQL    |
                              |            |
                              +------------+
```
