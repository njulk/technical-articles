Channel使用备注
-------------------

`Channel`的实现了“生产者－消费者”模型，`sender`将数据写入`buffer`(环形队列)，
然后`reciever`从`buffer`中取出数据.


```
+--------------+ copy to  +----------------+ copy to  +----------------+
|  Sender Queue+---------->   Data Buffer  +----------> Reciever Queue |
+--------------+          +----------------+          +----------------+

```
